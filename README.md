## ✨ Features Built

1. **🎨 Premium Modern Aesthetics**:
   - Palette inspired by royal indigo/violet and rose accents.
   - Beautiful custom illustrations for the **Hero** and **About** sections representing digital collaboration, mentorship, and literacy.
   - Clean card-based layout for initiatives and programs.

2. **🌓 Native Theme Toggle (Light / Dark Mode)**:
   - Synchronizes automatically with the user's OS preference (`prefers-color-scheme`).
   - Manual override toggle button in the header.
   - Prevents styling flash (FOUC) on load using a small inline script.
   - Persists the selected choice securely in `localStorage`.

3. **📱 Fluid Responsive Layout**:
   - Mobile-first CSS framework using modern Flexbox & CSS Grid.
   - Auto-collapsing hamburger menu for smaller devices.
   - Sticky navigation header that alters background transparency on scroll.

4. **✅ Smart Interactive Volunteer Form**:
   - Input field validations for Name, Email format, and Message.
   - Instant visual error styling (rose outlines) if a user attempts to submit incomplete inputs.
   - Dynamic slide-in validation success feedback ("*Form Submitted Successfully*") that resets the form and auto-clears.

5. **♿ Accessibility & Performance Optimized**:
   - Clean semantic HTML structure (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
   - Accessible descriptions on interactive button controls and images (`alt` attributes).
   - Dynamic text sizing using CSS `clamp()` for perfect readability on all resolutions.

## 🏃 How to Run the Project Locally

No installation or build steps are required. You can launch the site instantly:

### Option 1: Direct File Launch
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/she-can-foundation-web.git
   ```
2. Navigate to the directory and double-click `index.html` to open it in your default web browser.

### Option 2: Local Development Server
For the best experience (including clean path resolving and smooth scrolling), you can run a local server:
* **VS Code**: Install the **Live Server** extension, open the project folder, and click **"Go Live"** in the status bar.
* **Python (Terminal)**: Run `python3 -m http.server 8000` inside the folder and open `http://localhost:8000` in your browser.
* **Node.js**: Install `serve` globally via `npm i -g serve`, run `serve` inside the folder, and open `http://localhost:3000`.
